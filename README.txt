# MedVault

This is a simple healthcare project.

It allows users to:

* Login and Register
* Book appointments
* View dashboards (Patient / Doctor / Admin)

---

## How to run

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend/medvault
mvnw.cmd spring-boot:run
```

---

## Tech Used

* React (Frontend)
* Spring Boot (Backend)
